# Flowers CNN Classification 🌸

This repository contains a **Convolutional Neural Network (CNN)** model built with TensorFlow and Keras to classify different types of flowers.

## Tech Stack

- **Python 3.12**
- **TensorFlow & Keras**
- **Matplotlib** (For data visualization)
- **TensorFlow Datasets**

## Project Structure

- `cnn.py`: The main script where the model architecture is defined (including `Conv2D`, `MaxPooling2D`, `Flatten`, `Dense`, and `Dropout` layers) along with optimization callbacks (`EarlyStopping`, `ReduceLROnPlateau`, `ModelCheckpoint`).
- `requirements.txt`: The list of required Python libraries and dependencies for this project.

## Installation & Setup

After activating your virtual environment, install the required packages using:

```bash
pip install -r requirements.txt
```
